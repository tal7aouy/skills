# Issue Policy

> Maintained by Tal7aouy Developer Skills.

This project welcomes bug reports, feature requests, and other
feedback via GitHub issues. Use the templates provided under
`.github/ISSUE_TEMPLATE` when available.

For **sensitive information** such as security vulnerabilities or
private data exposure, please **do not** open a public issue. Instead
email the maintainers via the contact on the
[GitHub profile](https://github.com/tal7aouy) and include:

- Brief description of the issue
- Steps to reproduce, if applicable
- Any relevant logs or variables

Maintainers aim to acknowledge receipt within **48 hours** and
provide a status update within **7 days**.
